# Stellaris AI Helper

AI-powered assistant for asking questions about Stellaris, with live save file integration and game data extraction.

## About This Project

A personal project built over several iterations. The code was generated by AI and debugged through multiple versions. AI also created the button images and logo.

## Features

### AI-Powered Q&A
Ask questions about Stellaris and get accurate answers based on actual game data rather than AI hallucinations.

### Live Game Data
Connect the app to your Stellaris saves and ask questions about your current empire:
- "How many alloys do I have?"
- "What's my fleet power?"
- "What am I currently researching?"

### Dual Data Sources
- **Wiki Data**: Human-written descriptions and guides
- **Game Data**: Accurate stats extracted directly from game files

### Deep Search Mode
Find prerequisite chains for technologies - useful for "how do I unlock X?" questions.

### Conversation Mode
AI remembers previous questions for follow-up queries on complex topics.

## How It Works

1. You type a question
2. The app searches data files for relevant entries
3. Matching data is sent to the AI
4. The AI answers using that specific information

## Components

### Stellaris AI Helper (Main App)

The main application with:
- Question/answer interface
- Settings panel for API configuration
- History tracking
- Response caching
- Live save file monitoring

### Game Data Extractor

A separate tool to extract game data directly from your Stellaris installation. Produces clean JSON with:
- Technologies with costs, tiers, prerequisites
- Buildings with resources and modifiers
- Ship sizes and components
- Starbase modules and buildings
- Resources, traits, ethics, civics
- Traditions and ascension perks
- And more

## Installation

### Option 1: Run from Source

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the main app:

```
python stellaris_app.py
```

3. (Optional) Extract game data:

```
python stellaris_game_extractor.py
```

4. Enter your API key in Settings

### Option 2: Download the EXE

1. Go to [Releases](../../releases)
2. Download the latest`.exe` file
3. Run it - no installation needed
4. Enter your API key in Settings

## Requirements

- Python 3.8+ (if running from source)
- API key from a compatible AI provider
- Stellaris installation (for game data extraction)
- Non-Ironman saves (for live game data)

## Live Game Data

The app can read your actual save files to answer questions about your current game.

### How It Works

1. Enable "Live game data" checkbox
2. The app watches your Stellaris save directory
3. When you save in-game, it auto-loads the save
4. Ask questions about your empire

### Requirements

- Non-Ironman saves only (Ironman saves are encrypted)
- Save must be uncompressed (normal save, not ironman)

### Supported Data

From your save file, the app can see:
- Current date
- Empire name
- Resources (energy, minerals, alloys, etc.) with exact amounts
- Income and spending breakdown
- Fleet power
- Planet count and population
- Research output and current research

## Game Data Extraction

Use the extractor tool to pull data directly from your Stellaris installation.

### How to Use

1. Run`stellaris_game_extractor.py`
2. Select your Stellaris installation folder
3. Choose output location
4. Click "Extract All Data"
5. Place the output JSON in the`source data` folder

## Data Files

The app uses two optional data files:

| File | Source | Description |
|------|--------|-------------|
|`stellaris_unified_fixed.json`| Wiki | Human descriptions, guides |
|`stellaris_game_data.json`| Game files | Accurate stats, costs, prerequisites |

Both can be used together - the app will search both and label results appropriately.

## Modes Explained

### Single Question (Default)
One question → One response. AI doesn't remember previous questions. Best for quick queries, uses fewer tokens.

### Conversation Mode
AI remembers previous questions and answers. Good for follow-up questions or complex topics.

### Include Prerequisites (Deep Search)
Finds prerequisite chains - what technologies/items lead to others. Useful for "how do I unlock X?" questions. Uses more tokens.

### Live Game Data
When enabled, includes your current save file data in the AI context. Ask about your empire's resources, fleets, and research.

### Game Extracted Data
When enabled, searches the game-extracted data for accurate stats and costs.

## Known Limitations

- Search may occasionally return unexpected results
- Live data requires non-Ironman saves
- Game data extraction requires Stellaris installation
- Data may become outdated with new game patches
- Ironman saves cannot be read (encrypted)
- Doesnt supports mods yet
