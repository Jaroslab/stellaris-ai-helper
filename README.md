# Stellaris AI Helper

AI-powered assistant for asking questions about Stellaris.

## About This Project

This is a small personal project I built over a couple of days. I have limited coding knowledge, so the code was generated by AI and then debugged through several iterations to get everything working. The button and logo images were also created by AI.

It's not perfect, but it works. If you find bugs or want to improve it, feel free.

## Why This Tool Exists

AI models don't actually know precise details about Stellaris. They might give outdated information, hallucinate tech trees that don't exist, or mix up mechanics from different games.

This tool solves that by feeding the AI only relevant game data from the wiki dump. It keeps context focused so the AI can give accurate answers.

## How It Works

1. You type a question
2. The app searches the data file for relevant entries
3. Only matching data is sent to the AI
4. The AI answers using that specific information

## Modes Explained

### Single Question (Default)
One question → One response. AI doesn't remember previous questions. Best for quick queries, uses fewer tokens.

### Conversation Mode
AI remembers previous questions and answers. Good for follow-up questions or complex topics.

### Include Prerequisites (Deep Search)
Finds prerequisite chains - what technologies/items lead to others. Useful for "how do I unlock X?" questions. Uses more tokens.

## Included Data

The data file (`stellaris_unified_fixed.json`) contains information dumped from the Stellaris wiki:

- Technologies
- Buildings
- Other important game elements

**NOT included:**
- Species traits
- Mod-specific content
- Complete wiki coverage

**Note:** The JSON data may become outdated with new game patches. Feel free to update it yourself.

## Installation

### Option 1: Run from Source

1. Install dependencies:

pip install -r requirements.txt

2. Run the app:

python stellaris_app.py

3. Enter your API key in Settings

### Option 2: Download the EXE

1. Go to [Releases](../../releases)
2. Download the latest `.exe` file
3. Run it - no installation needed
4. Enter your API key in Settings

## Requirements

- Python 3.8+ (if running from source)
- API key from a compatible AI provider

## Known Limitations

- Search may occasionally return unexpected results
- Data does not include species traits or mod content
- Some wiki information may be missing

## Changes

Live Game Data-

The app can read your actual save files to answer questions about your current game.

How It Works-

1. Enable "Live game data" checkbox
2. The app watches your Stellaris save directory
3. When you save in-game, it auto-loads the save (auto saves got checked too)
4. Ask questions like "How many alloys do I have?" or "What's my fleet power?"

Requirements-

- Non-Ironman saves only (Ironman saves are encrypted)
- Save must be uncompressed (normal save, not ironman)
